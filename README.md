# 🛒 Shopping Cart — T-Shirt Store

Projeto de carrinho de compras desenvolvido com HTML, CSS e JavaScript durante o curso de Front-End Intermediário da *Faculdade Descomplica* .

O objetivo do projeto foi praticar conceitos intermediários de desenvolvimento front-end, com foco em manipulação do DOM, lógica de carrinho de compras e interatividade utilizando JavaScript puro.

# 📌 Sobre o projeto

A aplicação simula uma loja virtual de camisetas, permitindo ao usuário visualizar produtos, adicionar itens ao carrinho e acompanhar o valor total da compra de forma dinâmica.

O projeto foi desenvolvido para reforçar conceitos importantes utilizados no desenvolvimento de interfaces interativas para e-commerce.

# 🚀 Funcionalidades

✔️ Exibição de produtos

✔️ Adição de camisetas ao carrinho

✔️ Atualização dinâmica da quantidade de itens

✔️ Cálculo automático do valor total

✔️ Remoção de produtos do carrinho

✔️ Interface responsiva e interativa

✔️ Atualização em tempo real utilizando JavaScript

# 🛠️ Tecnologias utilizadas
- Tecnologia	Descrição
- HTML5	Estrutura da aplicação
- CSS3	Estilização e responsividade
- JavaScript	Lógica da aplicação e manipulação do DOM
